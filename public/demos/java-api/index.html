API Java com DB — CRUD + Testes

Demonstração estática do projeto de API REST desenvolvido com Java e Spring Boot, com foco em persistência, validação, migrations e documentação.

Java Spring Boot JPA Flyway Testcontainers OpenAPI

O que este projeto entrega

  • API REST com operações CRUD completas
  • Validação de entrada e tratamento padronizado de erros
  • Persistência com banco de dados e migrations via Flyway
  • Testes de integração mais próximos do ambiente real
  • Documentação de endpoints com OpenAPI / Swagger

Como usar

O projeto pode ser executado localmente para explorar endpoints, validar regras de negócio e consultar a documentação da API.
